## Overview

Project Ashwatthama is a modern, AI-powered web application that combines intelligent chatbot interactions and AI-generated images within a responsive, dark-themed interface. It’s designed to provide users with natural conversational experiences and quick, high-quality visual content generation.

## Key Features

- **AI Chatbot**: Built with Google’s Gemini API for real-time, natural language conversations.
- **Image Generator**: Utilizes Stability AI for fast and creative image generation from text prompts.
- **Persistent History**: Chat memory stored locally for seamless experience continuity.
- **Elegant Interface**: Polished dark mode UI with intuitive layout and iconography.
- **API Integration**: Secure integration with multiple third-party AI services.

## Technologies Used

- **Frontend**: React.js 18, React Router, Axios, React-Toastify, Lucide React, DOMPurify
- **Backend**: Express.js, CORS, Dotenv, Form-Data
- **AI Services**: Google Gemini API, Stability AI API
- **Utilities**: Concurrently
- **Styling**: Tailwind CSS, shadCN, Acernity UI
- **Deployment**: Vercel for frontend, Render for backend


## Outcome

This project deepened understanding of real-time AI integration, modular component design, and full-stack application scaling. The result is a smooth, feature-rich app that demonstrates the potential of AI-powered interactivity and creativity tools.

Project Ashwatthama is now live, providing users a platform to chat with an intelligent assistant and generate visual content instantly.
